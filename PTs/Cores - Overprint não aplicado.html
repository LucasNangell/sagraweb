Cores - Overprint não aplicado

Arquivo apresenta elementos gráficos sobre fundo colorido sem aplicação de Overprint. No material impresso existe a possibilidade do papel branco ficar aparente entre os elementos gráficos e o fundo colorido:

- Página 01: Moldura (Linhas e elementos em preto)

Providências
  • No software de diagramação, aplicar Overprint nos elementos apontados e gerar novo arquivo .pdf
Dicas

Overprint impede que seja criada a reserva de cor no elemento que está abaixo, ou seja, o elemento de cima não recorta o elemento de baixo.

Quando se aplica Overprint no elemento que está acima, ocorre a mistura de cores do elemento de cima com o que está embaixo.

Quando se remove o Overprint, o elemento de baixo é recortado obedecendo o formato do que está em cima, fazendo com que a cor de cima fique completamente visível.

Uso comum do Overprint

Em textos pretos sobre fundos coloridos, pra evitar que a reserva de cor abaixo do texto não crie filetes brancos no caso de problema de registro (encaixe das cores)

Em elementos coloridos sobre fundos coloridos claros, para criar um efeito de mistura dando a impressão de existir uma terceira na junção dos elementos

Quando não usar Overprint

Em elementos brancos, sobre fundos de cor inclusive branco, pois elemento irá desaparecer

Em elementos coloridos sobre fundos escuros, pois o elemento de cima irá desaparecer

Quando se deseja que o elemento de cima seja impresso na sua cor pura
