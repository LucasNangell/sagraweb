Sugerimos que a prova seja conferida com atenção, pois a CGraf não aceitará reclamações posteriores referentes a serviços que tenham tido a impressão autorizada.

Para autorizar a impressão responda este e-mail, sem alterar o campo Assunto:

- informando "Impressão autorizada conforme Prova; e
- indicando Nome e Número de Ponto.
Obs.: Quando houver mais de um modelo na prova encaminhada, indicar quais estão aprovados.

Para solicitar alterações responda este e-mail, sem alterar o campo Assunto:

- informando "Solicitamos que sejam feitas as seguintes alterações:" e detalhando as alterações necessárias; ou
- anexando um novo arquivo para impressão.
IMPORTANTE:

A Prova PDF é sempre disponibilizada para os Serviços da Cota de Papelaria. Para os demais serviços de impressão (OSs), a Prova PDF é disponibilizada apenas quando estiver fora das seguintes situações:

- Saída impressa estiver igual ao Original encaminhado pelo solicitante;
- Repetição de serviço sem alterações;
- Arquivo original encaminhado pelo solicitante no formato PDF.
